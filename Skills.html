Metacognition

When Coding you will almost always run into problems.
So I have metacognition in the way that i can figure out
what could be causing the problem and adapting the code
to make it work. Esports also uses this quite often.
In Esports the game state is always changing. This means
adapting on the fly to the new situation that happen on screen.

Critical Thinking

Coding requires alot of Critical Thinking because circumstances
change really easily. Being able to see when you should make
changes to what you are doing is very essential. Most Esports
are a team games. Very few being one on one. As a team you
have to be able to make descisions about the game quickly and
well.
